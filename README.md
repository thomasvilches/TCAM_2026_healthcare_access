# Social Inequality and Access to Healthcare:  
## An Analysis Using Unsupervised Machine Learning in Greater São Paulo

**Trends in Computational and Applied Mathematics**  
**Author:** Thomas Nogueira Vilches  
**Date:** 2026-01-15

This repository keeps the code that was used to perform the analyses presented in the paper __Social Inequality and Access to Healthcare: An Analysis Using Unsupervised Machine Learning in Greater São Paulo__

## Database

A relational database was built and is available in the file *database.sql* at https://drive.google.com/file/d/1PfeadRbk4zzgGuSbHBPh5HqVxIO5YbVM/view?usp=sharing

We recommend restoring the SQL file in a PostgreSQL server, e.g., using pgAdmin.
