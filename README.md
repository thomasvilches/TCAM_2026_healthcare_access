This repository keeps the code that was used to perform the analyses presented in the paper __Social Inequality and Access to Healthcare: An Analysis Using Unsupervised Machine Learning in Greater São Paulo__

## Database

A relational database was built and is available in the file *database.sql* available at https://drive.google.com/file/d/1PfeadRbk4zzgGuSbHBPh5HqVxIO5YbVM/view?usp=sharing

We recommend restore the SQP file in a PostgreSQL server, e.g., using PGAdmin.
